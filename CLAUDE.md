# CLAUDE.md - 프로젝트 규칙

## 주석 규칙

- 모든 Dart 소스 파일에는 **한국어 주석**으로 코드의 동작과 흐름을 설명해야 한다.
- 파일 상단에 `/// 파일 설명` 주석을 반드시 작성한다.
- 모든 클래스 위에 `/// 클래스 역할 설명` 주석을 작성한다.
- 모든 public 메서드/함수 위에 `/// 메서드 동작 설명` 주석을 작성한다.
- 복잡한 로직 블록(조건문, 반복문, 데이터 변환 등)에는 인라인 주석(`//`)으로 흐름을 설명한다.
- enum 값에는 각 값의 의미를 주석으로 설명한다.
- 위젯의 build 메서드 내부에서 UI 구성 요소마다 역할을 주석으로 설명한다.

## 주석 스타일 예시

```dart
/// 주식 종목 카드 위젯
/// - 종목명, 티커, 현재가, 변동률을 표시
/// - 탭하면 상세 화면으로 이동
class StockCard extends StatelessWidget {
  /// 카드 탭 시 상세 화면으로 네비게이션 처리
  void _onTap() {
    // 종목 상세 화면으로 이동
    Navigator.push(...);
  }
}
```

## 코드 작성 규칙

- 언어: Dart (Flutter)
- 테마 색상: 파란색 계열 (`0xFF1565C0`)
- 상승: 초록색 (`0xFF2E7D32`), 하락: 빨간색 (`0xFFC62828`)
- 추천 등급: 강력매수, 매수, 관망, 매도, 강력매도
- 시장: KOSPI, KOSDAQ, NYSE, NASDAQ
- 모든 금액 표시는 `formatters.dart`의 유틸 함수를 사용한다.
- 한국 원화는 `₩`, 미국 달러는 `$` 기호를 사용한다.
